## 项目介绍

CoderPro 是一款专为开发者设计的桌面端效率辅助工具，具备以下核心特性：

### 🎨 设计理念
- **苹果玻璃质感 x 硬核科技风**：极致的毛玻璃特效（Glassmorphism）、等宽字体、赛博青/荧光绿点缀色
- **全局无边框悬浮窗**：随时拖拽、不遮挡工作视线、保持置顶
- **本地 JSON 数据驱动**：无需联网，极简极速，保证数据私密

### 🎯 适用场景
- 📚 快速访问常用开发文档和技术网站
- 💻 保存和管理代码片段、AI 提示词
- ✅ 规划和管理每日开发任务
- 🚀 提升开发工作效率

## 快速开始

### 方式一：直接运行（推荐新手）

#### Windows 用户
1. 下载已编译的 `CoderProLauncher.exe` 启动器
2. 将启动器放置在项目根目录
3. 双击运行 `CoderProLauncher.exe`
4. 程序会自动安装依赖并启动

### 方式二：源码运行（推荐开发者）

#### 1. 环境准备
确保您的电脑已安装：
- **Node.js** (建议 v16 及以上版本)
- **npm** (随 Node.js 一起安装)
- **Git** (用于克隆代码)

可选：
- **Python 3.8+** (使用 Python 启动器)

#### 2. 获取代码
```bash
git clone https://github.com/YOUR_USERNAME/CoderPro.git
cd CoderPro
```

#### 3. 安装依赖
```bash
npm install
```

> 💡 **提示**: 如果安装 Electron 速度较慢，可以设置 npm 镜像：
> ```bash
> # Windows PowerShell
> npm config set ELECTRON_MIRROR https://npmmirror.com/mirrors/electron/
> 
> # 或使用 .npmrc 文件（项目中已包含）
> ```

#### 4. 启动应用
```bash
# 一条命令启动 Vite 前端服务和 Electron 主进程
npm run electron:dev
```

启动成功后，您会在桌面上看到：
- 主悬浮窗（赛博青色高光图标）
- 任务悬浮窗（右侧独立面板）

## 功能模块详解

### 1. 主悬浮窗（反应堆核心）

#### 基本操作
- **查看状态**：启动后，桌面会出现一个带有赛博青色高光的小型悬浮图标
- **拖拽移动**：鼠标按住图标即可在桌面任意位置拖拽（带有发光动画反馈）
- **展开/收起**：点击图标，悬浮窗将无缝展开为带有毛玻璃背景的二级菜单面板
- **关闭面板**：点击右上角"X"按钮或再次点击图标即可收起

#### 使用技巧
- 将悬浮窗放置在屏幕边缘，不遮挡主要工作区域
- 拖拽时会有视觉反馈，松手即固定位置
- 收起状态占用空间最小，适合专注工作时使用

---

### 2. 二级菜单（功能面板）

展开面板后，左侧为导航栏，包含三大核心模块：

#### 📌 网址记录 (URL Manager)

**功能说明**：
- 记录常用开发文档、技术网站或服务器链接
- 支持自定义名称和 URL
- 点击"打开"按钮直接调用系统默认浏览器访问

**操作步骤**：
1. 点击左侧导航栏的"网址记录"图标
2. 点击"添加网址"按钮
3. 输入网址名称和 URL 地址
4. 点击保存
5. 使用时点击"打开"按钮即可在浏览器中访问

**使用场景**：
- 快速访问常用文档（MDN、Stack Overflow 等）
- 保存项目相关的服务器地址
- 收藏常用的开发工具网站

---

#### 📝 文本记录 (Text/Snippet Manager)

**功能说明**：
- 专门用于保存代码片段 (Snippets) 或常用 AI 提示词 (Prompts)
- 支持横向滚动查看长文本
- 一键"复制"功能，快速粘贴到代码编辑器

**操作步骤**：
1. 点击左侧导航栏的"文本记录"图标
2. 点击"添加文本"按钮
3. 输入标题和文本内容（支持多行代码）
4. 点击保存
5. 使用时点击"复制"按钮即可复制到剪贴板

**使用场景**：
- 保存常用的代码片段（如 React 组件模板、SQL 查询等）
- 收藏 AI 提示词模板
- 记录常用的命令或配置片段

**使用技巧**：
- 使用有意义的标题，方便快速识别
- 长代码片段会自动支持横向滚动查看
- 复制后会有成功提示

---

#### ✅ 今日任务 (Task Manager)

**功能说明**：
- 详细规划每日待办任务
- 任务状态（完成/未完成）会实时同步到独立任务悬浮窗
- 支持任务的增删查改

**操作步骤**：
1. 点击左侧导航栏的"任务管理"图标
2. 点击"添加任务"按钮
3. 输入任务描述
4. 点击保存
5. 勾选任务复选框标记完成状态

**使用场景**：
- 规划每日开发任务
- 跟踪项目进度
- 管理待办事项

**使用技巧**：
- 每天早上添加当日任务清单
- 完成任务后及时勾选，保持成就感
- 任务状态会实时同步到独立悬浮窗，方便随时查看

---

### 3. 今日任务悬浮窗（独立面板）

#### 功能特点
- 启动程序时，会自动在屏幕右侧出现独立的"今日任务"透明面板
- 该面板可以被自由拖拽到任意位置
- 用于快捷预览今日任务进度
- 支持在面板内直接点击 Checkbox 标记完成
- 状态会实时写回本地 JSON 数据

#### 使用技巧
- 将任务面板固定在屏幕右侧，不影响主要工作区
- 随时查看任务进度，保持工作效率
- 快速勾选完成任务，无需打开主面板

## 数据存储

### 存储位置

所有数据均存储在操作系统级别的应用数据目录下：

| 操作系统 | 存储路径 |
|---------|---------|
| **Windows** | `C:\Users\用户名\AppData\Roaming\coderpro\coderpro_data.json` |
| **macOS** | `~/Library/Application Support/coderpro/coderpro_data.json` |
| **Linux** | `~/.config/coderpro/coderpro_data.json` |

### 数据格式

数据文件采用 JSON 格式，结构如下：

```json
{
  "urls": [
    {
      "id": 1,
      "name": "MDN Web Docs",
      "url": "https://developer.mozilla.org"
    }
  ],
  "texts": [
    {
      "id": 1,
      "title": "React Component Template",
      "content": "function Component() {\n  return <div>Hello</div>;\n}"
    }
  ],
  "tasks": [
    {
      "id": 1,
      "text": "完成用户登录功能",
      "completed": false
    }
  ]
}
```

### 数据备份

您可以定期备份 `coderpro_data.json` 文件，以防数据丢失：

```bash
# Windows PowerShell
Copy-Item "$env:APPDATA\coderpro\coderpro_data.json" "D:\Backup\coderpro_data_backup.json"

# macOS/Linux
cp ~/Library/Application\ Support/coderpro/coderpro_data.json ~/Backup/
```

## 常见问题

### Q1: 启动时提示"npm command not found"

**解决方案**：
1. 确保已安装 Node.js
2. 将 Node.js 添加到系统环境变量 PATH
3. 重启终端或 IDE 后重试

### Q2: Electron 安装速度慢或失败

**解决方案**：
```bash
# 设置 Electron 镜像
npm config set ELECTRON_MIRROR https://npmmirror.com/mirrors/electron/

# 或者使用 .npmrc 文件（项目中已包含）
```

### Q3: 应用窗口无法显示

**解决方案**：
1. 检查是否有多个显示器，窗口可能出现在其他屏幕上
2. 尝试重启应用
3. 查看任务栏是否有应用图标，右键选择"移动"

### Q4: 数据丢失或无法保存

**解决方案**：
1. 检查应用数据目录是否有写入权限
2. 查看 `coderpro_data.json` 文件是否被其他程序占用
3. 尝试以管理员身份运行应用

### Q5: 打包后的 exe 文件无法运行

**解决方案**：
1. 确保将 exe 文件放在项目根目录
2. 检查是否缺少依赖（node_modules）
3. 使用 Python 启动器重新打包：
   ```bash
   pip install pyinstaller
   pyinstaller --onefile --windowed --name CoderProLauncher main.py
   ```

## 开发指南

### 项目结构详解

```
CoderPro/
├── src/                    # 前端源代码目录
│   ├── components/        # Vue 组件
│   │   ├── MainWidget.vue      # 主悬浮窗组件
│   │   ├── TaskManager.vue     # 任务管理组件
│   │   ├── TextManager.vue     # 文本管理组件
│   │   └── UrlManager.vue      # 网址管理组件
│   ├── App.vue          # 根组件
│   ├── main.js          # Vue 应用入口
│   └── style.css        # 全局样式
├── build/               # 构建资源（如图标）
├── main.js              # Electron 主进程脚本
├── preload.js           # Electron 预加载脚本
├── main.py              # Python 启动器（可选）
├── index.html           # HTML 模板
├── package.json         # 项目配置和依赖
├── vite.config.js       # Vite 构建配置
├── tailwind.config.js   # Tailwind CSS 配置
└── postcss.config.js    # PostCSS 配置
```

### 本地开发流程

1. **克隆项目**
   ```bash
   git clone https://github.com/YOUR_USERNAME/CoderPro.git
   cd CoderPro
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发环境**
   ```bash
   npm run electron:dev
   ```
   此命令会同时启动：
   - Vite 开发服务器（端口 5173）
   - Electron 主进程

4. **开始开发**
   - 修改 `src/` 目录下的 Vue 组件
   - 热重载会自动更新应用界面
   - 查看控制台输出调试信息

### 构建生产版本

```bash
# 构建前端资源并打包 Electron 应用
npm run electron:build
```

构建成功后，会在 `dist_electron/` 目录下生成：
- Windows: `win-unpacked/` 目录（包含可执行文件）
- 安装包（如果配置了 NSIS）

### 自定义配置

#### 修改应用名称和 ID

编辑 `package.json`：
```json
{
  "build": {
    "appId": "com.yourname.coderpro",
    "productName": "YourAppName"
  }
}
```

#### 修改主题颜色

编辑 `src/style.css` 中的 CSS 变量：
```css
:root {
  --primary-color: #00ff88; /* 赛博青 */
  --glass-bg: rgba(255, 255, 255, 0.1);
}
```
